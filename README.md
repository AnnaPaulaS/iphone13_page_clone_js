# iphone13_page_clone_js
Este projeto, se trata do clone da página home do iPhone 13, com a funcionalidade de mudança na cor do iPhone + responsividade.

Desenvolvido no curso “Formação Front-end - HTML, CSS, JavaScript, React e +” do Hora de Codar, pela Udemy.
